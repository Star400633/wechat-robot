# WECHATY-ROBOT

此项目是一个基于wecahty构建的Node.js 应用。

### wechaty中文文档 <https://docs.chatie.io/wechaty>

## 快速开始

### 1. 下载代码
```sh
git clone https://github.com/Star400633/wechat-robot.git
cd wechat-robot
```

### 2. 安装依赖
> 注意: Wechaty 需要 Node.js 的版本 >= 10, 建议运行 `node -v` 进行确认

```sh
npm install
```

### 3. 运行机器人
```sh
node examples/starter-bot.js
```
